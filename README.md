# Fastband Core Distribution

This public repository hosts the signed Sparkle appcast and release archives used by Fastband Core's in-app updater.

Source code, development history, credentials, and signing keys are not stored here. Update artifacts are generated and verified from the private Fastband Core source repository before publication.

Current channel: **MVP / internal**

- Appcast: `appcast.xml`
- Release archive: GitHub Releases
- Updates: signed, user-controlled download/install/restart
